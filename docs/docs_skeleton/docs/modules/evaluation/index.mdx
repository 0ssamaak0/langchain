# Evaluation

LangChain has a number of evaluation chains implemented to help grade your model's behavior. It also contains base evaluator interfaces to support common use cases like:

- Grading output correctness against a ground truth reference
- Comparing outputs
- Evaluating the efficacy of an agent's tool usage
